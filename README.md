# Predictive Maintenance of Industrial Equipment Using IoT Sensor Data

This repository contains a complete final-year AI/ML project that focuses on predicting industrial equipment failures using synthetic IoT sensor data.

## Project Overview
Unexpected machine failures lead to costly downtime, expensive repairs, and potential safety hazards. This project aims to predict machine failures before they occur by analyzing continuous streams of IoT sensor data (temperature, vibration, pressure, current, etc.), enabling **predictive maintenance**.

The workflow includes:
1. **Data Generation:** Synthesizing realistically noisy condition-monitoring datasets.
2. **Model Training:** Training Machine Learning classifiers (Logistic Regression, Random Forest, XGBoost) to predict machine health.
3. **Streamlit Application:** A real-time web dashboard displaying the probability of machine failure along with actionable recommendations.

## How to Run the Project

### 1. Install Dependencies
Make sure you have Python 3.8+ installed. Then install the required libraries:
```bash
pip install -r requirements.txt
```

### 2. Generate Synthetic Data
Run the data generation script to synthesize 10,000 sets of sensor readings simulating 5 industrial machines.
```bash
python src/data_generation.py
```
*This will create `data/sensor_data.csv`.*

### 3. Train the Model
Run the model training script. This script compares various algorithms such as Random Forest and XGBoost. It will output the `best_model.pkl` based on the F1-Score metric.
```bash
python src/model_training.py
```
*This will create the `models/best_model.pkl` and `models/scaler.pkl`.*

### 4. Run the Streamlit Application
Launch the web interface for the predictive maintenance dashboard:
```bash
streamlit run app.py
```

## Dashboard Features
- **Real-Time Input:** Adjust sliders for Temperature, Vibration, RPM, Current, etc.
- **Machine Health Prediction:** Will show probabilities for "Healthy", "Warning", or "Critical".
- **Maintenance Recommendation:** Displays actionable guidelines based on real-time classification.
- **Historical Analysis:** Scatter plots of Machine behaviors (Normal vs Failures).
- **Feature Importance Tracking:** Horizontal Bar Chart demonstrating which indicators (e.g. Temperature, Load) are the primary drivers of failure prediction.
